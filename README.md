LogStorm
========

Prerequisites
-------------
* [Java 8](http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html)
* [Apache Maven](https://maven.apache.org/)

Development
-----------

    git clone git@github.corp.ebay.com:hchen9/logstorm.git
    cd logstorm
    mvn install
